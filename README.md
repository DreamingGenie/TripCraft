# MoveTrip - 이동 시간을 고려한 스마트 여행 일정 플래너

## 1. 프로젝트 소개
MoveTrip은 사용자가 관광지를 일정표에 배치하면 장소 간 이동 시간을 자동 계산하여 현실적인 여행 일정을 만들 수 있도록 돕는 국내 여행 플래너 서비스입니다.

## 2. 기획 배경
기존 관광 정보 서비스는 장소 조회 중심이지만, 실제 여행자는 장소 간 이동 시간과 일정 충돌 여부를 직접 계산해야 합니다. MoveTrip은 공공데이터 기반 관광지 정보와 지도/길찾기 API(Application Programming Interface, 응용 프로그램 인터페이스)를 결합해 이 문제를 해결합니다.

## 3. 핵심 기능
- 지역별 관광지 조회
- 카테고리별 관광지 조회
- 장소 기반 여행 일정 생성
- 장소 간 이동 시간 자동 계산*
- 실시간 공동 작업환경
- Hotplace 등록
- 여행 일정 공유게시판
- 공지사항
- 회원 관리

## 4. 핵심 사용자 흐름
지역 선택 → 관광지 조회 → 장소 상세 확인 → 일정표에 장소 배치 → 이동 시간 자동 계산 → 시간 충돌 확인 → 일정 저장 → 공유게시판에 공유

## 5. 사용 API
- 한국관광공사 국문관광정보 API
- 한국관광공사 관광사진정보 API
- OpenWeatherMap API
- 한국환경공단 전기자동차 충전소 API
- 한국천문연구원 출몰시각 API
- 지도/길찾기 API(TMAP 등)
