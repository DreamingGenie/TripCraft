### 커뮤니티 댓글 기능 추가

---

#### 배경

커뮤니티 상세 뷰에 댓글 섹션이 "준비 중" 상태로 플레이스홀더만 존재했다.

---

#### 구현 내용

**DB**

`post_comment` 테이블 신규 생성.

- 게시글 삭제 시 댓글 CASCADE 삭제
- 작성자 탈퇴 시 `member_id = NULL` — 댓글 내용 보존, '탈퇴한 사용자'로 표시

**백엔드 (신규 7개 파일)**

| 파일 | 역할 |
|------|------|
| `PostComment.java` | 댓글 도메인 |
| `CommentItem.java` | 댓글 응답 DTO (authorNickname, mine 포함) |
| `CommentCreateRequest.java` | 댓글 등록 요청 DTO (1000자 이하 유효성 검증) |
| `PostCommentMapper.java` / `PostCommentMapper.xml` | DB 조회·등록·삭제 |
| `CommentService.java` / `CommentServiceImpl.java` | 비즈니스 로직 |
| `CommentController.java` | REST API |

**API 엔드포인트**

| 메서드 | 경로 | 인증 | 설명 |
|--------|------|------|------|
| `GET` | `/api/posts/{postId}/comments` | 불필요 | 댓글 목록 |
| `POST` | `/api/posts/{postId}/comments` | 필요 | 댓글 등록 |
| `DELETE` | `/api/posts/{postId}/comments/{commentId}` | 필요 | 댓글 삭제 (본인·ADMIN) |

**프론트엔드**

- `post.js` — `commentApi` 추가 (list / create / delete)
- `CommunityView.vue` — 댓글 목록·입력폼·삭제 버튼 구현
  - `Ctrl+Enter`로 댓글 등록 가능
  - 비로그인 시 등록 시도하면 로그인 안내 토스트 표시
  - 본인 댓글에만 삭제 버튼 노출
- `community.css` — 댓글 섹션 스타일 추가
  - 댓글 목록·작성폼 박스 분리, 아바타+닉네임 좌우 배치, 시간 우측 정렬

### 댓글 코드 정리 (simplify)

기능 구현 후 코드 품질 점검(Reuse·Simplification·Efficiency·Altitude 4개 관점)을 진행하고 아래 항목을 수정했다.

| 항목 | 파일 | 내용 |
|------|------|------|
| 불필요한 DB 쿼리 제거 | `CommentServiceImpl` | `getComments`의 게시글 존재 확인 `SELECT` 제거 — 존재하지 않는 postId면 빈 목록 반환으로 충분 |
| 중복 검증 헬퍼 추출 | `CommentServiceImpl` | `createComment`·`deleteComment`에 중복된 게시글 존재 확인 → `validatePost()` private 헬퍼로 추출 |
| role 추출 위치 이동 | `CommentController` → `CommentServiceImpl` | Controller에서 `Authentication.getAuthorities()` 직접 파싱하던 로직을 Service 내 `currentUserIsAdmin()` 헬퍼로 이동. Controller는 `memberId`만 전달 |
| 미사용 쿼리 제거 | `PostCommentMapper` / `.xml` | 서비스에서 호출하지 않는 `countByPostId` 제거 |
| API 병렬 호출 | `CommunityView.vue` | `openPost`에서 게시글 상세·댓글 목록을 순차 조회 → `Promise.all` 병렬 조회로 변경 |
| 에러 토스트 패턴 통일 | `CommunityView.vue` | `deleteComment`의 에러 처리를 `e.status === 401 ? '로그인이 필요합니다.' : '...'` 패턴으로 통일 |
