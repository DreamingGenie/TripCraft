- `TourApiSyncServiceImpl.java` — 17개 시도 × 6개 콘텐츠 타입 배치 수집
- `AttractionSyncController.java` — `POST /api/admin/attractions/sync` (전체/부분)
- `AttractionMapper.xml` — `api_created_at` INSERT 누락 수정
