# 캡스톤 1.0 아카이브 (as-submitted)

> ⚠️ **동결(frozen) 디렉터리 — 수정 금지.**
> SSAFY 11기 자율 프로젝트 TripCraft의 **팀(전진·송정기) 최종 제출 상태**를 그대로 보존한다.
> 코드 기준선은 `v1.0-capstone` 태그. 이후 1인 개발은 `master`의 Living 문서(`docs/*`)에서 진행하며,
> 이 폴더의 내용은 역사적 산출물로 갱신하지 않는다.

## 구성

| 경로 | 내용 |
|------|------|
| `설계문서/` | 제출 정본 설계문서 (요구사항·유스케이스·클래스·ERD·WBS·화면설계·기타·API·기술심화 2종 + 통합본 + PDF) |
| `out/` | 제출 PDF 8종 |
| `ai-report/` | AI 사용 보고서 |
| `최종완료보고서.md` · `제출체크리스트.md` | 최종 보고서·체크리스트 |
| `발표/` | 발표 자료(html·대본·스크린샷). 발표 PDF/PPTX 정본은 `out/` |
| `planning-notes/` | 기획·설계 과정 메모(요구사항/WBS 초안·리디자인·제안서·도메인 플랜 등) |
| `logs/` | 개발 일지 |

> 데모 영상(`demo.mp4`, 63MB)은 저장소 경량화를 위해 제외했다. GitHub Release 에셋/외부 링크 참조.
