### ODsay transit_cache 저장 실패 수정

- `OdsayClient.extractMode()` 반환값이 한국어 문자열(`"버스"`, `"지하철"`)이었던 것을 DB ENUM 호환 영문(`BUS`, `SUBWAY`, `WALK`)으로 수정
  - MySQL ENUM INSERT 실패가 서비스 레이어 try-catch에서 조용히 삭제되어 캐시가 저장되지 않던 문제 해결
  - 프론트엔드 `ScheduleView.vue`에 `TRANSPORT_DISPLAY` 맵 추가하여 영문 → 한국어 표시

### 후보군 삭제 401 오류 수정 (GlobalExceptionHandler)

- `ResponseStatusException(CONFLICT)` 발생 시 Spring Boot 기본 오류 처리가 `/error`로 재디스패치되는데, `/error`가 `anyRequest().authenticated()` 에 막혀 401 반환되던 문제 수정
  - `global/exception/GlobalExceptionHandler.java` 추가 (`@RestControllerAdvice`)
    - `ResponseStatusException` → `ApiResponse.fail(reason, statusCode)` JSON 직접 반환
    - 일반 `Exception` → 500 반환
  - `SecurityConfig`에 `.requestMatchers("/error").permitAll()` 추가 (안전망)
