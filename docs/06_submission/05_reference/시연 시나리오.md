# 시연 대본 & 시연 시나리오 — TripCraft

> 계획부터 공유까지, 여행 플래너 트립크래프트 — 화면 시연용 발표 대본

| 항목 | 내용 |
|------|------|
| 대상 | 프로젝트 심사위원·평가단 |
| 형식 | 1인 단독 시연 |
| 길이 | 약 3분 |
| 흐름 | 탐색 → 일정 → 협업 → 커뮤니티 → 마이페이지 |

## 시연 전 준비 (사전 세팅)

- 실시간 협업 시연용 일정(예: **부산 여행 · 4명**)에 동행자를 **미리 추가**해 둘 것
- '가져오기' 시연용으로 **공개된 여행 이야기 글 1건** 미리 확보
- 방문 지도에 **방문·예정 지역** 데이터가 채워져 있는지 확인

## 시연 시나리오 & 대본

| 단계 | 화면 · 조작 (시나리오) | 시연 대본 (그대로 읽기) |
|------|----------------------|------------------------|
| **01. 오프닝 · 랜딩** | • 랜딩 페이지 표시<br>• "회원가입 없이 탐색 가능" 영역 짚기<br>• 우측 상단 **로그인** 클릭 | 안녕하세요. 여행의 계획부터 공유까지 한곳에서 해결하는 여행 플래너, '트립크래프트'입니다. 보시는 랜딩 화면에서는 회원가입 없이도 관광지 탐색과 커뮤니티 둘러보기가 가능합니다. 본격적인 기능을 보여 드리기 위해, 먼저 로그인하겠습니다. |
| **02. 로그인 · 회원가입** | • 로그인 화면 진입<br>• **카카오로 로그인** 클릭<br>• 여행 작업실로 진입 | 먼저 회원가입을 진행해보겠습니다. 회원가입에는 닉네임, 이메일, 비밀번호를 입력해야 합니다. 이메일 가입과 더불어 카카오 간편 로그인을 지원합니다. 이번에는 카카오 로그인으로 빠르게 시작해 보겠습니다. |
| **03. 관광지 탐색** | • 상단 **탐색** 진입<br>• 검색어 입력 → 결과 확인<br>• 지도와 목록 함께 보기 | 로그인하면 여행 작업실로 들어옵니다. 상단 '탐색'에서는 전국의 관광지와 음식점, 숙박을 검색할 수 있습니다. 가고 싶은 장소를 검색해 보겠습니다. |
| **04. 상세 · 챗봇 · 담기** | • 관광지 카드 클릭 → 상세정보<br>• AI 챗봇으로 추천받기<br>• **담기** 또는 **일정 추가** | 검색 결과를 누르면 상세 정보를 확인할 수 있고, AI 챗봇에게 여행지 추천을 받을 수도 있습니다. 마음에 드는 장소는 보관함에 '담기'하거나, 일정에 바로 추가합니다. |
| **05. 일정 작업실 구성** | • **드래그 앤 드롭**으로 블럭 배치<br>• 블럭 이동 · 삭제<br>• **지도**로 동선 확인<br>• 이동수단 변경 (이동시간 자동 계산)<br>• **내 장소** 직접 추가 → 일정에 배치 | 담은 장소들은 일정 작업실에서 드래그 앤 드롭으로 원하는 날짜와 시간에 배치합니다. 블럭은 자유롭게 옮기거나 삭제할 수 있고, '지도'를 열면 하루 동선이 한눈에 보입니다. 장소 사이의 이동수단과 이동 시간은 자동으로 계산되며, 이동수단도 직접 바꿀 수 있습니다. 검색에 없는 장소는 '내 장소'로 직접 추가해 일정에 함께 넣습니다. |
| **06. 실시간 협업 · 공유** | • **공유** → 닉네임·이메일로 동행자 추가<br>• 여러 명 동시 편집 시연<br>• 링크 접근을 **읽기 전용**으로 설정 → 링크 전송<br>• 미리 협업해 둔 일정으로 결과 확인 | 트립크래프트의 핵심은 실시간 협업입니다. '공유'에서 닉네임이나 이메일로 동행자를 초대하면, 여러 명이 하나의 일정을 동시에 편집할 수 있습니다. 링크 접근을 '읽기 전용'으로 설정하면, 받은 사람은 일정을 보기만 할 수 있습니다. 이번엔 편집 권한이 부여된 협업 화면에서 팀원과 협업하는 모습을 보여드리겠습니다.  |
| **07. 여행 이야기 · 가져오기** | • **여행 이야기** → 글 상세 진입<br>• 좋아요 · 북마크 · 댓글<br>• **일정 그대로 가져오기** (시작일 선택)<br>• 가져온 일정 확인 | 완성한 일정은 '여행 이야기'에 공유됩니다. 다른 사용자의 글에는 좋아요와 북마크, 댓글을 남길 수 있고, 마음에 드는 일정은 '그대로 가져오기'로 내 일정에 그대로 담깁니다. 가져온 일정까지 확인해 보겠습니다. |
| **08. 글쓰기 · 수정** | • **새 여행 이야기** 작성<br>• 제목 · 내용 · 대표 사진<br>• 일정 연결 · 공개 범위 선택 → 등록<br>• 작성한 글 수정 | 새 글 작성도 간단합니다. 제목과 내용, 대표 사진을 넣고 내 일정을 연결한 뒤, 공개 범위를 정해 등록합니다. 작성한 글은 언제든 다시 수정할 수 있습니다. |
| **09. 마이페이지 · 방문 지도** | • **방문 지도** → **시도** 보기<br>• 시도 클릭 → 사진 위치 조정<br>• **시군구** 토글로 상세 확인 | 이제 마이페이지입니다. '방문 지도'는 다녀온 지역을 사진으로 채워 보여 줍니다. 시도를 클릭해 사진 위치를 조정할 수 있고, '시군구'로 전환하면 더 자세한 단위로 방문 기록을 확인할 수 있습니다. |
| **10. 마이페이지 · 관리 항목** | • **내 여행** — 카드 · 달력 보기<br>• **내 정보** → **내 장소** → **내가 쓴 글**<br>• **북마크 · 좋아요** 순으로 가볍게 | 이 밖에도 '내 여행'은 카드와 달력으로, '내 정보'와 '내 장소', '내가 쓴 글', 그리고 북마크와 좋아요까지 — 나의 여행 활동을 한곳에서 관리할 수 있습니다. |
| **11. 클로징** | • 화면을 작업실/랜딩으로 정리<br>• 발표 마무리 인사 | 이상으로, 계획부터 공유까지 함께하는 여행 플래너 트립크래프트의 시연을 마치겠습니다. 감사합니다. |
